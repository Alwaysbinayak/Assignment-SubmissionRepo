# User Directory (React)

This is a self-contained React project implementing the User Directory (Task 2) as requested.
It uses an internal mock API (no external network calls) so it works offline and behind firewalls.

## How to run

1. Extract the zip
2. Open a terminal inside the project folder
3. Run `npm install`
4. Run `npm start`

Open http://localhost:3000 in your browser.
